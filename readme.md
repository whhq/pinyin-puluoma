# 普逻码拉丁拼音文字

小写为拼音，大写第一位是部首，大写第二位是顺序排位，常用字在前，难字在后。大写第三、四位为L的是避开同码的罕用字。  



* *niV haoN，shiHB jieT!*  
* *你好，世界！*
